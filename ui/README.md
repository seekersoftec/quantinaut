# UI

For the Web channel.

Inspired by: https://github.com/karanpratapsingh/HyperTrade/

## Guide

Install pnpm:
```bash
    npm i -g pnpm
```

Install packages:
```bash
    pnpm i 
```


Run:
```bash
    pnpm run dev
```
